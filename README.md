## Counting and  Counter example (POC for RabbitMQ)

PoC for RabbitMQ on NodeJs :Docker-compose build dockerfile for each service (sender and receiver)

##### note : wait 20s for starting rabbitmq.

### Usage

```bash
docker-compose up
```

### Management Page

http://localhost:1567
